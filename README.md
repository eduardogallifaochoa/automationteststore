This is a personal Playwright automation project (JavaScript) using automationteststore.com to automate 3 user stories.

Automated flows

Flow #1 — Login
As a shopper, I want to log in, so that I can access my account.

Flow #2 — Add to cart
As a shopper, I want to search for a product, open its details page, and add it to the cart, so that I can prepare a purchase.

Flow #3 — Guest checkout
As a guest shopper, I want to checkout with items in my cart, so that I can place an order without creating an account.

## How to run

1. Clone the repo and open it in VS Code

2. Install dependencies:

```
npm i
npx playwright install
```

3. Run tests:
```
npx playwright test
```
(Optional) Run headed:
```
npx playwright test --headed
```
